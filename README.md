# Petrichor — TAM Lie Detector

> Interactive diagnostic that tests whether your addressable market is a real calculation or a story you tell investors.

A Claude Code skill that runs the TAM Integrity Test interactively — interrogating your market sizing math across five dimensions, tracing every number back to its source, and rebuilding your TAM/SAM/SOM from the bottom up when the foundation doesn't hold. Part of the [Petrichor Intelligence Lab](https://petrichorgrowth.com/intelligence-lab).

## Install

```
claude install github:petrichorprojects/petrichor-tam-lie-detector
```

## Run

```
/tam-lie-detector
```

## What you'll produce

- `TAM-INTEGRITY-SCORECARD.md` — Scores across all 5 integrity dimensions (Source, Sales Reality, Competitive Displacement, Budget Constraint, Timing)
- `MARKET-REALITY-REPORT.md` — Full diagnostic findings with specific evidence gaps identified
- `CORRECTED-TAM.md` — Bottom-up TAM reconstruction where original numbers don't hold
